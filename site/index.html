01 How it works

Connect sources, resolve dependencies, execute with one state model.

Viaduct keeps discovery, dependency mapping, planning, and execution in the same control plane so operators do not have to hand off context between tools.

01

Connect

Point Viaduct at your hypervisors. Connectors handle VMware, Proxmox, Hyper-V, KVM, Nutanix, and Veeam. Inventory normalizes into one model.

02

Assess

Dependencies resolve automatically - network, storage, and backup relationships rendered as a graph. Know what moves together before anything moves.

03

Migrate

Build a plan, dry-run it, get approval, execute with rollback safety. Same state whether you use the CLI, API, or dashboard.

02 Built for operators

Infrastructure workflows with calm defaults and explicit control.

The surface area stays compact, but the important details stay visible: normalized inventory, approvals, reporting, shared state, and rollback posture.

01

One model, every platform

Normalized inventory from VMware, Proxmox, Hyper-V, KVM, Nutanix, and Veeam. No per-vendor mental models.

02

Full audit trail

Every discovery, plan, approval, and execution logged. Export audit-ready reports when compliance asks.

03

Three interfaces, one state

CLI for automation, API for integration, dashboard for visibility. All read and write the same data.

04

Rollback safety

Dry-run simulations and approval gates before anything moves. Roll back if something goes wrong.
