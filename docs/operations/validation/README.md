# Validation Kit

This directory contains the working templates that support the Step 7 real-user validation round for Viaduct.

Use these files alongside [../real-user-validation-plan.md](../real-user-validation-plan.md).

## Files

- [Pre-session checklist](pre-session-checklist.md)
- [Participant note template](participant-note-template.md)
- [Round synthesis template](round-synthesis-template.md)

## Intended Use

1. Use [../real-user-validation-plan.md](../real-user-validation-plan.md) to recruit the right participants, run the session, and keep the scope narrow.
2. Use [pre-session-checklist.md](pre-session-checklist.md) before every session so the scenario stays comparable across participants.
3. Use [participant-note-template.md](participant-note-template.md) during each session.
4. Use [round-synthesis-template.md](round-synthesis-template.md) after all five sessions to turn observations into backlog decisions.

These templates are deliberately narrow. They are for the mixed-estate assessment and supervised pilot workflow Viaduct is currently optimizing, not a general product-research program.
