# Phase 3 Roadmap

Milestone target: end of January 2027

Phase 3 turns Viaduct into a lifecycle platform for long-running multi-hypervisor operations.

## Primary Outcomes
- support warm migration through block replication and cutover coordination
- add lifecycle capabilities such as cost modeling, drift detection, and policy evaluation
- expand backup portability workflows
- add KVM/libvirt and Nutanix AHV connector coverage
- introduce multi-tenancy and a connector SDK for ecosystem growth

## Planned Workstreams
1. Warm migration replication engine
2. Warm migration cutover coordinator
3. Cost modeling
4. Policy engine
5. Drift detection
6. Backup portability
7. KVM/libvirt connector
8. Nutanix AHV connector
9. MSP multi-tenancy
10. Community connector SDK
11. Lifecycle dashboard views
12. End-to-end lifecycle testing

## Definition of Done
- warm migration paths are available for supported platforms
- lifecycle insights help operators manage mixed estates after migration
- connector coverage expands beyond VMware and Proxmox
- the project can support both direct users and managed service providers
